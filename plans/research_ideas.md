## Part 1: 데이터 활용 아이디어

### 아이디어 1: Deep Learning 적용

**현재 논문의 접근**
- Ridge regression (선형 모델)

**제안**
```
뇌 활동 패턴 → [CNN/Transformer] → 감정 예측
```

**기대 효과**
- 비선형 관계 포착
- 복셀 간 공간적 관계 활용
- 더 높은 예측 성능

**필요 사항**
- [ ] PyTorch/TensorFlow 환경 설정
- [ ] 3D CNN 또는 Graph Neural Network 구현
- [ ] 과적합 방지 전략 (데이터 증강, dropout)

---

### 아이디어 2: 개인차 연구

**현재 논문의 접근**
- 5명의 평균 패턴 분석
- 개인차는 부차적으로만 언급

**제안**
```
개인별 뇌 패턴 → 성격/정서조절능력 예측
```

**연구 질문**
- 감정 표상 패턴이 개인마다 다른가?
- 이 차이가 성격 특성과 관련 있는가?
- 정서조절 능력이 높은 사람은 다른 패턴을 보이는가?

**필요 사항**
- [ ] 개인별 모델 학습
- [ ] 성격/정서 설문 데이터 (있다면)
- [ ] 개인차 통계 분석 방법 학습

---

### 아이디어 3: 시간 역학 분석

**현재 논문의 접근**
- 비디오 전체 → 하나의 fMRI 패턴
- 정적(static) 분석

**제안**
```
시간 t1 → t2 → t3 ... 에 따른 감정 변화 추적
```

**연구 질문**
- 감정은 시간에 따라 어떻게 변화하는가?
- 감정 전이(예: 공포→안도)의 신경 기제는?

**필요 사항**
- [ ] fMRI 시계열 데이터 접근
- [ ] Sliding window 분석
- [ ] 동적 연결성(dynamic connectivity) 방법론

---

### 아이디어 4: 크로스-모달 일반화

**현재 논문의 접근**
- 비디오(시각) 자극만 사용

**제안**
```
비디오 → 음악 → 텍스트
    ↓        ↓       ↓
  동일한 감정 표상인가?
```

**연구 질문**
- 시각적 공포와 청각적 공포는 같은 뇌 패턴?
- 모달리티 독립적인 감정 표상이 존재하는가?

**필요 사항**
- [ ] 다른 모달리티 데이터셋 탐색
- [ ] 공통 감정 공간(common emotional space) 구축 방법

---

### 아이디어 5: 감정 예측 BCI

**장기 목표**
```
실시간 fMRI → 감정 상태 디코딩 → 피드백
```

**응용**
- 정서 장애 환자 치료 (neurofeedback)
- 감정 조절 훈련
- 인터페이스 개인화

## Part 2: 데이터 한계점 분석

### 한계점 1: 적은 피험자 수 (N=5)

| 측면 | 문제점 |
|------|--------|
| 통계적 검정력 | 개인차 분석 어려움 |
| 일반화 | 5명이 인류 대표? |
| 재현성 | 다른 표본에서도 동일할까? |

**보완 방안**
- 다른 fMRI 데이터셋과 통합 분석
- 메타 분석 수행
- 공개 데이터셋 활용 (HCP, OpenNeuro)

---

### 한계점 2: 생태학적 타당성

| 측면 | 문제점 |
|------|--------|
| 환경 | fMRI 스캐너 = 좁고 시끄러움 |
| 자극 | 수동적 비디오 시청 |
| 맥락 | 실험실 vs 실제 생활 |

**보완 방안**
- 이동식 fNIRS/EEG 활용
- 가상현실(VR) 자극
- 경험표집법(ESM)과 결합

---

### 한계점 3: 레이블 불일치

```
크라우드 워커가 평가한 감정 ≠ 피험자가 실제 느낀 감정
```

| 문제 | 영향 |
|------|------|
| 주관성 | 같은 비디오, 다른 감정 |
| 개인차 미반영 | 피험자 본인의 경험 무시 |
| 문화 차이 | 평가자 vs 피험자 배경 |

**보완 방안**
- 피험자 본인의 감정 보고 수집
- 다차원 평가 (강도, 확신도 등)
- 생리적 지표(심박, 피부전도) 활용

---

### 한계점 4: 서양 중심 편향

```
34개 카테고리 = 영어권 문화 기반

누락된 감정:
- 한국: 정(情), 한(恨)
- 일본: 아와레(哀れ), 와비사비
- 독일: Schadenfreude
```

**보완 방안**
- 다문화 감정 분류 체계 개발
- 문화 비교 연구 수행
- 보편 감정 vs 문화 특수 감정 구분

---

### 한계점 5: fMRI 한계

| 측면 | 문제점 |
|------|--------|
| 시간 해상도 | ~2초 (감정 변화보다 느림) |
| 공간 해상도 | mm 수준 (뉴런 수준 X) |
| 비용 | 고가 장비, 접근성 제한 |

**보완 방안**
- 고해상도 fMRI (7T)
- 멀티모달 (fMRI + EEG)
- 침습적 기록 (환자 데이터)
