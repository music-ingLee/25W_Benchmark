이 문서는 논문을 본격적으로 읽기 전에 전체 그림을 파악하기 위한 가이드입니다.

--------------------------------------------------------------------------------
1. 한 줄 요약
--------------------------------------------------------------------------------
"감정은 뇌에서 고차원적, 카테고리적, 분산적으로 표상된다"

풀어서 말하면:
- 고차원적: 2개 차원(valence, arousal)이 아니라 27개 이상의 감정이 구분됨
- 카테고리적: "기쁨", "공포" 같은 카테고리가 차원보다 뇌 활동을 더 잘 설명
- 분산적: 특정 감정 = 특정 뇌 영역 (X) → 여러 영역의 네트워크가 함께 작동


--------------------------------------------------------------------------------
2. 실험 설계 (이것만 기억해!)
--------------------------------------------------------------------------------
자극:       2,185개 감정 유발 비디오 (무음)
피험자:     5명
측정:       whole-brain fMRI
레이블:     크라우드 워커들이 각 비디오 평가
            - 34개 감정 카테고리 (joy, fear, disgust...)
            - 14개 정서 차원 (valence, arousal...)


--------------------------------------------------------------------------------
3. 세 가지 분석 방법 (논문의 핵심!)
--------------------------------------------------------------------------------

[1] Neural Decoding (디코딩)
    뇌 활동 → 감정 예측
    "이 뇌 활동을 보고 어떤 감정인지 맞출 수 있나?"

[2] Voxel-wise Encoding (인코딩)
    감정 → 뇌 활동 예측
    "이 감정이 뇌의 어디를 활성화시키나?"

[3] Unsupervised Modeling
    레이블 없이 뇌 활동 패턴 분석
    "뇌 활동이 자연스럽게 어떤 클러스터를 형성하나?"


--------------------------------------------------------------------------------
4. 주요 발견 4가지
--------------------------------------------------------------------------------

발견 1: 분산 표상
        감정은 특정 영역이 아니라 여러 영역 네트워크에 표상됨
        (예: 공포 → 편도체만? X → 여러 영역이 함께)

발견 2: 카테고리 > 차원
        368/370 뇌 영역에서 카테고리 모델이 차원 모델보다 우수
        amygdala에서도 마찬가지!

발견 3: Transmodal 영역에 집중
        감정 정보는 고차 연합 영역(Default Mode Network)에서 주로 인코딩
        Visual → Semantic → Emotion 순서의 처리 계층 존재

발견 4: 클러스터 + 그라디언트
        뇌 활동이 감정별 클러스터 형성
        BUT 관련 감정(fear-horror) 사이에는 연속적 그라디언트

7. 모르는 용어가 나오면
--------------------------------------------------------------------------------

Voxel          = 3D 픽셀, fMRI의 최소 단위
BOLD           = 뇌 활동 측정 신호 (혈류 변화)
HCP360         = 뇌를 360개 영역으로 나눈 지도
Ridge          = 과적합 방지하는 회귀 방법
Cross-validation = 모델 성능 평가 방법
UMAP           = 고차원 데이터를 2D로 시각화하는 방법
Transmodal     = 여러 감각 정보가 통합되는 고차 뇌 영역
DMN            = Default Mode Network, 휴식 시 활성화되는 네트워크
